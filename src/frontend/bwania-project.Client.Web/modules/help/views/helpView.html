 Bwania c'est quoi ?

Bwania c'est quoi ?
Bwania est une plateforme de formation et d’auto-formation née du concours Waridlab édition 2013. Il a deux objectifs essentiels :

  • Etre un outil pour l’éducation et la formation en donnant les moyens aux utilisateurs de créer une base documentaire riche en contenu local.
  • Etre un pont entre l’école et l’entreprise.

Comment s'inscrire ?

Documents

 Qui peut consulter un document ?

Qui peut consulter un document ?
Tout le monde peut consulter un document sur la plateforme Bwania. Il suffit de créer un compte pour vous permettre de se connecter.

Comment consulter un document ?

Comment consulter un document ?

  • Cliquer sur le menu Formation ;

  • Cliquer sur détails pour avoir plus d'informations sur le document ;

  • Cliquer sur l’icône du document ou sur le nom du document pour lancer son téléchargement ;

Qui peut publier un document ?

Qui peut publier un document ?
Tout le monde peut publier un document sur la plateforme Bwania. Il suffit d’avoir un compte pour vous permettre de se connecter.

Comment publier un document ?

Comment publier un document ?

  • Pour publier un document ouvrez la liste déroulante ‘Participez’ puis cliquer sur ‘Ajouter un document’ ;

  • Remplissez les différents champs ;

  • Sélectionner le fichier à télécharger sur le serveur ;

  • Valider les informations ;

Comment supprimer un document ?

Comment supprimer un document ?
Pour supprimer un document il suffit de…

  • Accéder à son menu;
  • Accéder au menu publication;
  • Chercher le document à supprimer;
  • Cliquer sur le document à supprimer;
  • Supprimer le document;
Comment commenter un document ?

Comment commenter un document ?
Pour commenter un document il suffit de

  • Accéder à son menu;
  • Accéder au menu publication;
  • Chercher le document à commenter;
  • Accéder à la page pour poster un commentaire et l'éditer;
  • Valider et poster son commentaire;
